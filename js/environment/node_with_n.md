# nvm vs n

Node의 버전을 관리해주는 대표적인 도구로 nvm 과 n이 있다. 두 개의 프로젝트는 성격이 조금 다른데, 그 차이는 다음과 같다.

- nvm
    - 쉘 스크립트로 구성되어 있음
    - 명령어를 통해 자유롭게 버전을 오고갈 수 있음
- n
    - <https://github.com/tj/n>
    - Node 모듈
    - 원하는 버전을 쉽게 설치할 수 있음

# n 설치하기

```bash
# homebrew를 통해 n을 설치한다
brew install n

# 최신 버전의 node를 설치한다
sudo n lts
```
